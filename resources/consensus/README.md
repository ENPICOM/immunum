# Consensus Files

Generated by `scripts/generate_consensus.py` in the [immunum-scripts](https://github.com/thijsvanloef/immunum-scripts) repo.

## How they're generated

The script takes IMGT-numbered sequences (from CSV or IMGT-aligned FASTA) and computes per-position statistics:

1. **Amino acid frequencies** — counts residues at each IMGT position across all sequences, keeping AAs (and gaps) above a 5% frequency threshold
2. **Occupancy** — fraction of sequences with a residue at each position
3. **Region assignment** — maps each position to its IMGT region (FR1–FR4, CDR1–CDR3)
4. **End cleaning** (optional) — removes gap entries at sequence edges and renormalizes frequencies to occupancy 1.0

## Output format (CSV)

| Column          | Description                                        |
|-----------------|----------------------------------------------------|
| `position`      | IMGT position number                               |
| `consensus_aas` | Pipe-separated AAs sorted by frequency descending  |
| `frequencies`   | Corresponding pipe-separated frequencies           |
| `occupancy`     | Fraction of sequences with a residue here          |
| `region`        | IMGT region (FR1, CDR1, FR2, CDR2, FR3, CDR3, FR4)|

## Source
derived from IMGT-aligned RepSeqIO (version 1.9) germline Human and Mouse VJ genes.
